# wikipedia-indexer

## How to run an application
* You need to have a java installed on the machine where you want to run this application, as it is based on scala
* Usage: `java -jar WikipediaIndexer-0.1.0.jar <ARTICLES_FILE> <INDEX_DIR>`
* `<ARTICLES_FILE>` = path to the wikipedia articles file
* `<INDEX_DIR>` = directory to store indexes
